
# EriBooks – Coming Soon Landing Page

EriBooks is a web platform designed to connect Eritrean authors with readers around the world. This repository contains the official **Coming Soon landing page** for the EriBooks project.

 **Live Site:** https://eribooks.org

---

##  About the Project

EriBooks aims to create a centralized platform where:

* Readers can discover and explore Eritrean books
* Authors can showcase their work and connect with their audience
* Users can access both free and premium content

This landing page is deployed to introduce the project while the full application is under development.

---

##  Features (Phase -1 )

*  Book discovery (search by title, author, category)
*  Author profiles with detailed information
*  Direct communication with authors
*  Free-to-read book section
*  User authentication and personalized experience
*  Global access for readers and authors
